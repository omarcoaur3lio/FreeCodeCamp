ESP-IDF

ESP-IDF Espressif IoT Development Framework é um conjunto de programas, compiladores e bibliotecas para o desenvolvimento de aplicações voltadas à internet das coisas.

O framework foi criado pela empresa ESPRESSIF, e esta sendo continuamente melhorado e aumentado, com inúmeros exemplos de aplicações, aproveitando praticamente todos os recursos de hardware e software do ESP32. A linguagem usada como base é a linguagem C.

Material necessário

Para tirar o máxmo de proveito dessa documentação e já inicializar os primeiros projetos com um ESP32, é sugerido o seguinte material:

  • Uma placa ESP32
  • Um cabo USB - USB A / micro USB B
  • Um computador rodando Linux, MacOS ou Windows
