ESP-IDF

ESP-IDF Espressif IoT Development Framework é um conjunto de programas, compiladores e bibliotecas para o desenvolvimento de aplicações voltadas à internet das coisas.

O framework foi criado pela empresa ESPRESSIF, e esta sendo continuamente melhorado e aumentado, com inúmeros exemplos de aplicações, aproveitando praticamente todos os recursos de hardware e software do ESP32. A linguagem usada como base é a linguagem C.

Material necessário

Para tirar o máxmo de proveito dessa documentação e já inicializar os primeiros projetos com um ESP32, é sugerido o seguinte material:

  • Uma placa ESP32
  • Um cabo USB - USB A / micro USB B
  • Um computador rodando Linux, MacOS ou Windows

Configuração do ambiente de desenvolvimento

Existe mais de uma forma de configurar o ambiente para desenvolvimento com ESP-IDF. Na próxima seção serão apresentaremos duas delas, denominadas instalação manual e instalação via VSCode.

Instalação

Está seção é destinada a orientar o processo de instalação do framework ESP-IDF em diferentes sistemas operacionais. A instalação pode ser feita de duas formas:

A instalação via VSCode é frequentemente mais facilitada que a instalação manual, isso porque são utilizados scripts que automatizam o processo de instalação das demais ferramentas utilizadas pelo framework.

Instalação manual

Escolha abaixo para qual plataforma deseja realizar a instalação do framework.

Instalação das dependências

Para compilar com ESP-IDF, você precisa obter os seguintes pacotes. O comando a ser executado depende de qual distribuição do Linux você está usando:

Sistemas de base Debian (Ubuntu, mint, etc..)

sudo apt-get install git wget flex bison gperf python3 python3-pip python3-setuptools cmake ninja-build ccache libffi-dev libssl-dev dfu-util libusb-1.0-0

Arch Linux

sudo pacman -S --needed gcc git make flex bison gperf python-pip cmake ninja ccache dfu-util libusb

Obtendo o ESP-IDF

Para desenvolver aplicativos para o ESP32, você precisa das bibliotecas de software fornecidas pelo Espressif no repositório ESP-IDF. Para obter ESP-IDF, navegue até o diretório de instalação desejado e clone o repositório.

mkdir -p ~/esp cd ~/esp git clone --recursive https://github.com/espressif/esp-idf.git

Configurando as ferramentas

Além do ESP-IDF, você também precisa instalar as ferramentas usadas pelo ESP-IDF, como compilador, depurador, pacotes Python, etc.

cd ~/esp/esp-idf ./install.sh

Configure as variáveis ​​de ambiente

As ferramentas instaladas ainda não foram adicionadas à variável de ambiente PATH. Para tornar as ferramentas utilizáveis ​​na linha de comando, algumas variáveis ​​de ambiente devem ser definidas. ESP-IDF fornece outro script que faz isso.

. $HOME/esp/esp-idf/export.sh

Hello World

Agora você está pronto para preparar seu aplicativo para ESP32. Você pode começar com o projeto “hello_world” do diretório de exemplos no IDF.

cd ~/esp cp -r $IDF_PATH/examples/get-started/hello_world .

Configure o projeto

cd ~/esp/hello_world idf.py set-target esp32 idf.py menuconfig

Gere uma build do projeto

idf.py build

Grave o projeto no ESP

idf.py -p PORT [-b BAUD] flash

Onde:

  • PORT: é o endereço da entrada USB onde o ESP está conectado. (algo como /dev/ttyUSB0).
  • -b BAUD: É um parâmetro opcional. Indicará qual baud rate deve ser utilizado durante a gravação.

Arduino como um componente

É possível também utlizar o core do Arduino como um componente para o ESP-IDF. Esse método possibilita a utilização da bibliotecas do Arduino diretamente no código IDF.

Para preparar o projeto basta seguir os passos abaixo:

  • Baixe e instale o esp-idf
  • Crie um projeto IDF em branco (a partir de um dos exemplos)
  • Na pasta do projeto, crie uma pasta chamada components e clone o repositório esp32-arduino-lib-builder
mkdir -p components cd components git clone https://github.com/espressif/arduino-esp32.git arduino cd arduino git submodule update --init --recursive cd ../.. make menuconfig

Para mais informações, acesse o repositório do projeto.
